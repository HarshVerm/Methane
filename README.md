# Methane

Methane is a clone of linkedin.

## Installation

Clone the repo https://github.com/wahed2102/Methane.git.
cd into to "cd Methane"

```bash
npm install
```

## Usage

```bash
npm start
json-server --watch db.json --port 8000
```
